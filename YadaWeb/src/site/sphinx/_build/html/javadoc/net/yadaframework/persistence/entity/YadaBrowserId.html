YadaBrowserId¶

public class YadaBrowserId implements Serializable¶

This class uniquely identifies the user browser. It can be used to prevent multiple submissions when the user is not authenticated (no login performed). Identification can either come from a cookie, from local storage, or from a number of “browser leaks” that, used together, can produce a “best guess” about the browser identity. See https://browserleaks.com/
