YadaSqlBuilder¶

public class YadaSqlBuilder implements CloneableFiltered¶

Classe di utilit? che costruisce una stringa sql o jpql partendo dagli elementi che la compongono, opzionalmente presenti. Si utilizza istanziandola con new() visto che non può essere un singleton.

See also: YadaSql
