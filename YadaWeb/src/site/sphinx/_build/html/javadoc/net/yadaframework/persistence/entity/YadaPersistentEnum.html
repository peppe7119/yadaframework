YadaPersistentEnum¶

public class YadaPersistentEnum<E extends Enum<E>>¶

Needed to store a localized enum in the database, on which to perform localized search and sort operations. To retrieve the localized value call YadaPersistentEnum.getLocalText() For every Enum class in Java there is a number of rows in the database holding the enum class name and its ordinals. For each such row there are a number of rows holding the localized values. E.g. the “ProcessState” enum with values “RUNNING” and “STOPPED” produces the following rows:

Table YadaPersistentEnum (logical view)

en_US Executing
