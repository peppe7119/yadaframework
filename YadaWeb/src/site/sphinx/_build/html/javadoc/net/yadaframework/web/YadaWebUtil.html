modalConfirm¶

public String modalConfirm(String message, String confirmButton, String cancelButton, Model model, Boolean reloadOnConfirm, Boolean openModal)¶

Show the confirm modal and optionally reloads when the confirm button is pressed

Parameters:
  • message – text to be displayed (can be null for default value)
  • confirmButton – text of the confirm button (can be null for default value)
  • cancelButton – text of the cancel button (can be null for default value)
  • model –
  • reloadOnConfirm – (optional) when true, the browser will reload on confirm, adding the confirmation parameter to the url
  • openModal – (optional) when true, the modal will be opened. To be used when the call is not ajax
