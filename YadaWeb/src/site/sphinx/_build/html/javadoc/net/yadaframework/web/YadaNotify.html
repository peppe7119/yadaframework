YadaNotify¶

public class YadaNotify¶

Build a notification dialog to be shown on the response page or after a redirect. More than one notification can be saved for the same request: all messages will be shown in one dialog. A single instance can be reused after changing its parameters. Multiple instances can be used too. The last method called on the instance must be yadaSave() to store the configuration values. Example:

return YadaNotify.instance(model).yadaOk()
    .yadaTitle("Info Accepted").yadaMessage("Thank you for your support")
    .yadaAutoclose(2000).yadaReloadOnClose().yadaSave();
