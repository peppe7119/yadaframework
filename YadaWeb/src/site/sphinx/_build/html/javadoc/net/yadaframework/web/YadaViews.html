YadaViews¶

public interface YadaViews¶

Some view strings that have specific effects when returned from a @Controller

Fields¶

AJAX_CLOSE_MODAL¶

String AJAX_CLOSE_MODAL¶

Close any open modal

AJAX_NOTIFY¶

String AJAX_NOTIFY¶

Open a notification modal when returning from an ajax request. The notification must have been saved using the YadaNotify API before returning.

AJAX_REDIRECT¶

String AJAX_REDIRECT¶

Ajax method causes a browser redirect. Set the “targetUrl” attribute in the model.

AJAX_REDIRECT_NEWTAB¶

String AJAX_REDIRECT_NEWTAB¶

Set this attribute to true to open the redirect page in a new tab

AJAX_REDIRECT_URL¶

String AJAX_REDIRECT_URL¶

The name of the attribute to set with the full target redirect url

AJAX_REDIRECT_URL_RELATIVE¶

String AJAX_REDIRECT_URL_RELATIVE¶

The name of the attribute to set with the relative target redirect url: it will be put inside @{ }

AJAX_RELOAD¶

String AJAX_RELOAD¶

Perform a page reload when returning from ajax method

AJAX_SERVER_ERROR¶

String AJAX_SERVER_ERROR¶

When returning from the ajax method, a “server error” modal is shown. A custom error description can be added as a model attribute named “errorDescription” before returning.

AJAX_SUCCESS¶

String AJAX_SUCCESS¶

Ajax method returns cleanly.
