makeWebDriver¶

public WebDriver makeWebDriver(File customProfileDir, InetSocketAddress proxyToUse, Set<Cookie> cookiesToSet, int driverType)¶

Create a new browser instance positioning the window

Parameters:
  • customProfileDir – the folder where to store the user profile, can be null to use the default temporary profile. The folder is created when missing.
  • proxy –
  • cookiesToSet – cookies to set after the first get of a document. Can be null or empty. Cookies are set only when a cookie with the same name has not been received. It’s not possible to set cookies BEFORE the first get (by design of WebDriver).
  • driverType – DRIVER_FIREFOX, DRIVER_CHROME
Throws:
  • MalformedURLException –
