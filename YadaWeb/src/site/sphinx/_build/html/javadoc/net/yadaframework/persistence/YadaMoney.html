YadaMoney¶

public class YadaMoney¶

An amount of money with a 1/10000 precision, stored as a long both in java and in the database. “The rule of thumb for storage of fixed point decimal values is to store at least one more decimal place than you actually require to allow for rounding.” https://stackoverflow.com/q/224462/587641 Some world currencies use 3 decimals: http://www.thefinancials.com/Default.aspx?SubSectionID=curformat
