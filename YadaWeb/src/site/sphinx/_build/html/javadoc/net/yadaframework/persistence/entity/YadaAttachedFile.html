YadaAttachedFile¶

public class YadaAttachedFile¶

A “pointer” to a file that has been uploaded into the “contents” folder. When a file is associated to an object, an instance of this class is created. Files can still exist after the object has been deleted, and can be re-attached to many objects using different titles, sort orders etc.
