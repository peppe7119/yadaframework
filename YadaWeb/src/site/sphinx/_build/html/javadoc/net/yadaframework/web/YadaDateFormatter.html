public class YadaDateFormatter implements Formatter<Date>¶

La formattazione automatica della data usa il formato html5 “yyyy-MM-dd” indipendentemente dal locale, altrimenti la cosa non può funzionare. Ho però aggiunto un metodo

printLocal¶

public String printLocal(Date object, Locale locale)¶

Metodo di utilit? da usare per convertire una data in una stringa formattata come specificato in messages.properties (quindi locale-dependent). Si può usare in pagina così: th:text=”${@yadaDateFormatter.printLocal(story.publishDate, #locale)}” Però non è molto meglio dello standard thymeleaf th:with=”df=#{date.format}” th:text=”${#calendars.format(story.publishDate, df)}”
