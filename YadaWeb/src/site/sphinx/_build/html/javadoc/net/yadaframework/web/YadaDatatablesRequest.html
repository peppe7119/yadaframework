getLength¶

public int getLength()¶
Returns:Number of records that the table can display in the current draw. It is expected that the number of records returned will be equal to this number, unless the server has fewer records to return. Note that this can be -1 to indicate that all records should be returned

public int getStart()¶
Returns:Paging first record indicator. This is the start point in the current data set (0 index based - i.e. 0 is the first record).

getYadaSql¶

public YadaSql getYadaSql()¶

Can be used to set join and where conditions on the query. Use “e” as the alias for the root table. E.g.: “e.user.role=4”
