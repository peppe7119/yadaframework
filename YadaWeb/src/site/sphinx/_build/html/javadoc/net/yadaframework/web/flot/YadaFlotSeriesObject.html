YadaFlotSeriesObject¶

public class YadaFlotSeriesObject extends YadaFlotSeriesOptions implements Serializable¶

A Flot Series in Object format. JSON Data format for Data:

{
    color: color or number
    data: rawdata
    label: string
    lines: specific lines options
    bars: specific bars options
    points: specific points options
    xaxis: number
    yaxis: number
    clickable: boolean
    hoverable: boolean
    shadowSize: number
    highlightColor: color or number
}

See also: <A href=”https://github.com/flot/flot/blob/master/API.md” target=”_blank”>Flot Documentation</A>

setXaxis¶

public void setXaxis(Integer xaxis)¶

The “xaxis” and “yaxis” options specify which axis to use. The axes are numbered from 1 (default), so { yaxis: 2} means that the series should be plotted against the second y axis.

Parameters:
