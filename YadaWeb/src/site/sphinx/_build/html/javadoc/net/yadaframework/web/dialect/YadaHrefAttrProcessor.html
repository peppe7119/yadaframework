YadaHrefAttrProcessor¶

public class YadaHrefAttrProcessor extends AbstractAttributeTagProcessor¶

Questa classe aggiunge il tag yada:href per il calcolo di un url tipo /res-0001/js/a.js dove 0001 è calcolato dinamicamente e coincide con il numero di build. Questo permette di evitare una sintassi tipo: th:href=”@{|${res}/css/bootstrap.min.css|}” dove “res” è un valore iniettato da un HandlerInterceptorAdapter e usare invece yada:href=”@{/res/css/bootstrap.min.css}” Tutorial: http://www.thymeleaf.org/doc/html/Extending-Thymeleaf.html
