getLocalValue¶

public String getLocalValue(Object entity, String attributeName, Locale locale, Boolean... returnNull)¶

Get the localized value of an entity’s attribute

Parameters:
  • entity – the entity instance
  • attributeName – the name of the attribute that must be defined as Map
  • locale – the locale for the value, use null for the current request locale
  • returnNull – (optional) true if null must be returned when no value is present instead of the empty string
Returns:

the localized value in the specified locale, or in the default configured locale if the value is not set for that locale, or an empty string if no value is found or null if no value is found and returnNull is true.
