isYadaLocalEnum¶

public boolean isYadaLocalEnum(Object someClass)¶

Check if a class is a YadaLocalEnum. For example YadaJobState is a YadaLocalEnum. Used by list.html to check for enum options.

Parameters:
  • someClass – the class to check (or a non-class object, in which case the result will be false)
Returns:

true if the class implements YadaLocalEnum
